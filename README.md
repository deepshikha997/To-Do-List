### Instructions to run on local machine (Windows)
    First Clone the project 
    npm i package.json
    
(if it shows vulnerability or error run command)

    npm audit fix

Run command 

    npm start

The website is running at localhost:3000

### Dependencies
**Backend**
1. NodeJs
2. ExpressJS
3. Body Parser
4. MongoDB
5. Mongoose

**Frontend**
1. Extend JavaScript (EJS)
2. Bootstrap 4
3. JavaScript
